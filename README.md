首先这是一个 antd umi 项目，结合路由缓存实现Tabs标签页效果。
1. 标签页可缓存页面；
2. 双击标签可刷新页面；
3. 不分页大数据列表；
4. 自定义样式的富文本框；
5. 表格列排序，支持个性化保存；

后期将结合项目实际需要陆续加入各种组件，如：
1. 拖拽布局；
2. 可拖拽Modal；
3. 可最大化的Modal；
4. ................
