# cala-react
我的第一个react项目，不断学习更新中！

首先这是一个 antd umi 项目，结合路由缓存实现Tabs标签页效果。
1. 标签页可缓存页面；
2. 双击标签可刷新页面；
3. 不分页大数据列表；
4. 自定义样式的富文本框；
5. 表格列排序，支持个性化保存；
6. ......

示例访问地址： http://admin.caladog.com 临时帐号密码：admin/admin
